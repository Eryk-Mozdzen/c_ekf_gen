# EKF implementation in C

- based of CMSIS-DSP
- auto-generated docs in LaTeX

# Prerequisites

1. Create venv for Python:
```bash
python3 -m venv venv
```
2. Source venv:
```bash
source ./venv/bin/activate
```
3. Install dependencies using pip:
```bash
pip3 install -r requirements.txt
```

# Usage

Call on of the example python scripts in order to generate files for EKF in C.

```bash
python3 example1.py
```
